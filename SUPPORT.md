# Support ComfyUI Universal FP8 / NVFP4 Quantizer

This project is free, open-source software. Sponsorship is optional and never
required to install, run, or update the node.

## About the maintainer

I am a South Korea-based, self-taught independent developer whose formal
education ended with high school. I currently maintain this project without
institutional or corporate backing and without a stable income from software
development. Sponsorship gives me more time to test large models, investigate
compatibility issues, and turn experiments into documented tools that remain
freely available to the ComfyUI community.

## Sponsor development

You can sponsor ongoing development through
[GitHub Sponsors](https://github.com/sponsors/thepororo).

Sponsorship helps fund:

- compatibility testing against new ComfyUI and comfy-kitchen releases;
- validation across supported diffusion-model architectures;
- reproducible documentation and troubleshooting guides;
- maintenance, bug fixes, and requested architecture support.

Sponsors do not receive a functionally restricted build. When practical,
development previews, roadmap notes, and example workflows may be shared with
sponsors before their general release.

## Bugs and feature requests

For reproducible bugs and generally useful feature requests, use
[GitHub Issues](https://github.com/thepororo/ComfyUI-NVFP4-Quantizer/issues).
Do not include private models, API keys, credentials, or personal information.

## Paid implementation support

Installation help, model-specific workflow tuning, and custom architecture
support are separate from sponsorship. Open an issue with a non-confidential
outline first. Scope, availability, and commercial terms must be agreed before
work begins. Sponsorship does not guarantee individual support or a delivery
date.

# ComfyUI Universal FP8 / NVFP4 Quantizer 후원 및 지원

이 프로젝트는 무료 오픈소스 소프트웨어입니다. 후원하지 않아도 노드를
설치하고 실행하고 업데이트할 수 있습니다.

## 개발자 소개

저는 한국에서 활동하는 고졸 독학 개발자입니다. 현재 기관이나 회사의 지원,
안정적인 개발 수입 없이 이 프로젝트를 독립적으로 유지하고 있습니다. 후원은
대형 모델을 테스트하고 호환성 문제를 조사하며, 실험 단계의 결과물을 누구나
무료로 사용할 수 있는 문서화된 도구로 발전시키는 시간을 확보하는 데 직접
도움이 됩니다.

## 개발 후원

[GitHub Sponsors](https://github.com/sponsors/thepororo)를 통해 지속적인
개발을 후원할 수 있습니다.

후원금은 다음 작업에 사용됩니다.

- ComfyUI 및 comfy-kitchen 새 버전 호환성 테스트
- 지원하는 확산 모델 아키텍처별 검증
- 재현 가능한 문서와 문제 해결 가이드 작성
- 유지보수, 버그 수정, 요청받은 아키텍처 지원

후원 여부에 따라 노드 기능을 제한하지 않습니다. 가능한 경우 개발 미리보기,
로드맵 소식, 예제 워크플로를 정식 공개 전에 후원자에게 제공할 수 있습니다.

## 버그 및 기능 요청

재현 가능한 버그와 일반 기능 요청은
[GitHub Issues](https://github.com/thepororo/ComfyUI-NVFP4-Quantizer/issues)에
등록해 주세요. 비공개 모델, API 키, 인증 정보, 개인정보는 올리지 마세요.

## 유료 구현 지원

설치 지원, 모델별 워크플로 최적화, 맞춤 아키텍처 지원은 후원과 별도입니다.
기밀이 아닌 요청 개요를 이슈로 먼저 등록해 주세요. 작업 시작 전 범위, 가능
여부, 비용 조건을 합의합니다. 후원은 개별 지원이나 특정 납기일을 보장하지
않습니다.
